How Does it Work?

The program enables seamless communication between hardware sensors and a central server. It uses MQTT (Message Queuing Telemetry Transport), a lightweight publish-subscribe network protocol, to transfer data from the hardware to the server in real-time. The host for MQTT is HiveMQ, a scalable and reliable MQTT broker that ensures data delivery. The incoming data is in JSON format, which is parsed and visualized on a web-based platform, displaying graphical representations like line charts or bar graphs to help analyze the sensor readings.
